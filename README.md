# babbelen
Chat client for the webengineering course at the DHBW Stuttgart

## Features
- Required
  - Authentication
  - List of chat-rooms
  - Send messages
  - Display messages
    - Display old messages
    - Live update
    - Support smileys (display smiley as image)
    - List and color users in room
  - Responsive design
  - Accessible
- Optional
  - Search
  - End to end encryption
  - Notifications
